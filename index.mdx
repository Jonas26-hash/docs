## How it works

<Steps>
  <Step title="Register and log in">
    Create a user account via `POST /auth/register`, then call `POST /auth/login` to receive a JWT token.
  </Step>
  <Step title="Authenticate requests">
    Include your token in every request: `Authorization: Bearer <your-token>`. Tokens are valid for 24 hours.
  </Step>
  <Step title="Use the API">
    Call any endpoint through the API Gateway at `http://localhost:8080`. All services are accessible through a single entry point.
  </Step>
  <Step title="Manage the full care lifecycle">
    Register patients, schedule appointments, issue prescriptions, dispense medications, and process billing — all from one unified API.
  </Step>
</Steps>
